# 碳化硅探测器快速仿真软件辐射效应电场模拟的研究



## 研究背景

项目拟选取4H-SiC探测器为研究对象，开展基本电学性能实验测试；开展探测器性能与辐照剂量的变化研究，获得其量效关系曲线；完善模拟软件电场分布的模拟，为理解辐照损伤提供依据。研究结果可为探测器抗辐照加固设计提供数据支撑，具有重要的理论价值和应用前景。

## 拟解决的关键科学问题

瞬间电流技术（Transient Current Technique ，TCT）利用半导体结构中非平衡自由载流子运动在电极上产生的信号，可以在实验上用来探测材料的性质，或者识别更复杂的结构，还可以用来确定硅探测器电场分布。利用电离辐射或激光在反向偏压探测器内产生的电子空穴对的漂移和扩散。在电场影响下，载流子漂移并在读出电极中产生电流。电流脉冲包含有关漂移电荷量及其迁移速率的信息。根据光照方向，TCT可分为 top-TCT和 edge-TCT 两种技术。在信号产生的瞬间，我们可以认为载流子受到的非平衡力只有电场力，假设载流子初始速度近似为0，在刚开始运动的时刻可近似看作加速度恒定的加速运动，qE=mv/t，则v=qEt/m，载流子的迁移速率与此处的电场大小成正比。信号上升的上升沿近似为一次函数，积分面积与信号的大小成正比，因此载流子的迁移速率与信号积分面积成正比。

在模拟方面，以现有RASER框架为基础，结合上述获得电场分布的理论，开发基于GEANT4的4H-SiC探测器电场分布功能的实现。不需要完全求解泊松方程和连续性方程，而是使用优化的第三方有限元方法库在探测器的网格结构上求解拉普拉斯和泊松方程，用于提供半导体探测器内部载流子动力学的快速模拟。主要目标是用edge-TCT模拟辐照前后探测器中的电流，可用于理解激光下探测器响应、模拟edge-TCT测量、计算漂移速率、电荷收集和电场分布。随后加入辐照模型，参考辐照后探测器电场分布的测试结果，优化辐照模型以及模拟方法，实现RASER预测辐照后探测器电场分布的目标。
