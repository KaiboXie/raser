# 石墨烯优化的抗辐照碳化硅低增益雪崩探测器研究

[[王聪聪]]

- [ ] 研究背景

由于碳化硅的电子亲和能和表面态密度制约，高温退火（>1000 ℃）形成的欧姆接触电阻率较高、稳定性差、工艺可重复性低；且高温退火会激活碳化硅中杂质导致电信号的散射，制约其信噪比、电荷收集、时间分辨和工作稳定性的提升。此外，大尺寸碳化硅探测器（探测边长大于500μm）电荷收集信号的不均匀性，即信号电流的畸变会造成定时误差，进而影响器件的时间分辨的提高。本项目提出研制石墨烯优化的碳化硅低增益雪崩抗辐照探测器（即石墨烯****/4H-SiC LGAD****），解决高温欧姆接触和不均匀的电荷收集导致的碳化硅探测器时间性能下降的科学问题。本项目在研究过程中设计基于石墨烯优化的低增益雪崩碳化硅探测器结构，优化工艺流程，制作低温欧姆接触石墨烯****/4H-SiC LGAD****抗辐照探测器；通过****理论和实验相结合优化时间分辨模型，定量研究内部电场和外延结构对时间分辨影响的物理机制。
- [ ] **研究内容**

**1)**      **石墨烯****/4H-SiC**LGAD****粒子探测器****外延结构设计

石墨烯/4H-SiC LGAD探测器外延结构中增益层的设计尤为关键。研究增益层掺杂浓度、厚度与击穿电压与工作区间的物理机制，实现探测器稳定的增益效果。

**2)**      **石墨烯****/4H-SiC LGAD****粒子探测器制备**

·         **低温欧姆接触**

研究欧姆接触电阻率与石墨烯层数、欧姆接触层的有效掺杂浓度、电极金属类型，金属厚度，退火温度和退火时间的物理机制，实现低温欧姆接触，提高器件性能。

·         **制作终端结构**

由于增益层的高电场区域靠近器件表面而导致的边界效应使得器件更容易提前击穿，需要设计合理的刻蚀终端结构，避免器件提前击穿。具体研究刻蚀端面质量、刻蚀深度和角度等不同刻蚀终端结对器件击穿电压的影响。

**3)**      **石墨烯****/4H-SiC LGAD****粒子探测器性能研究**

·         **电学性能**

根据石墨烯/4H-SiC LGAD探测器电流-电压（I-V）和电容-电压（C-V）特性，研究探测器击穿电压、全耗尽电压、漏电流和有效掺杂浓度。

·         **时间分辨率**

石墨烯/4H-SiC LGAD粒子探测器的时间分辨主要由粒子的朗道噪声、沉积能量变化、器件电场均匀性、信噪比和电子学抖动决定。粒子的朗道噪声、沉积能量变化主要由器件有源层的厚度决定；器件电场均匀性与石墨烯插入层有关；信噪比主要与器件工艺质量和材料缺陷有关。本项目拟通过实验与理论量化分析时间分辨与器件外延结构各项参数、石墨烯层数、退火温度的关联，理解各项参数造成时间分辨变化的物理机制。
