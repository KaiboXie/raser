# Si strip simulation 


角度与位置分辨                                             

## Introduction 
计算探测器的位置分辨，需要先对触发事例进行筛选，当探测器的某一通道读出大于4倍电子学噪声时，定义其为seed通道，然后判断该通道相邻的通道信号是否大于1.5倍电子学噪声，以此类推，将所有cluster事例提取出。然后利用重心法对其击中位置进行重建，最后计算残差的均方根值RMS作为衡量探测器位置分辨能力的参数。
定义归一化位置分辨参数K为探测器位置分辨的仿真结果与二进制读出位置分辨的比值$$K=\frac{RMS}{\frac{pitch}{\sqrt{12}}}(ADC的分辨率为12bit)$$
在探测器的实际应用中，绝大多数粒子并非垂直入射到探测器，而是有一定的角度。随着粒子夹角的增加，信号被多个条带收集的概率增高，进而可以更精确的重建粒子的径迹。但随着夹角的进一步增大，分布在每个条带上的信号更容易被电子学噪声影响，反而劣化了探测器的位置分辨能力。即存在一个最佳位置分辨。
