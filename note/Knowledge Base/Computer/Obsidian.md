## 图像保存 

 参考： https://post.smzdm.com/p/az6mokvo/ 
 
1.  安装插件 : Image auto upload Plugin
2. 安装 PicGo: https://picgo.github.io/PicGo-Doc/
	- 设置： 图床设置 / 腾讯云COS （SecretKey 请咨询： [[史欣]] ) 
![PicGo-2.3.1-raser-setup.png](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/PicGo-2.3.1-raser-setup.png)

- 如上图设置好后，点击“确定” 和“设为默认图床”按钮： ![PicGo-Setup-Button.png](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/PicGo-Setup-Button.png)

- 如需开机自启，可在“PicGo设置”里选择“开机自启”： 
![PicGo-Startup.png](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/PicGo-Startup.png)
