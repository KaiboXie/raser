# Design and Fabrication of 4H-SiC Sam-APD Ultraviolet Photodetector 
https://ieeexplore.ieee.org/document/8246019
---
# Abstract

本文设计了一种具有表面刻蚀窗口的4H-SiC sam APD结构。通过蚀刻深台面来制造器件端子，同时生长40nm的台面边缘抗反射层和窗口钝化层。结果表明，在260nm波长下，器件的响应度峰值可达0.15A/W，量子效率为71.5%，击穿电压为127V，器件的光学增益接近1000。此外，我们发现抗反射层可以提高响应度，台面边缘的钝化可以将器件的泄漏减少1~2个数量级。

# Design and Fabrication

![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-1_structure.png)

1.设计：

| 层 | 厚度 | 掺杂浓度 | 目的 |
| : - : | : - : | : - : | : - : |
| $P^+$ | 0.3 μm | 5e19 $cm^{-3}$ | 实现理想的欧姆接触 |
| $N$ | 0.25 μm | 4e17 $cm^{-3}$ | 调节电场强度，提供足够的光生载流子 |
| $N^-$ | 1 m | 1e15 $cm^{-3}$ | 获得高量子效率和低击穿电压 |
| $N$ Buffer | 1 μm | 1e18 $cm^{-3}$ | 提高外延质量，调节击穿电压 |

1m的N吸收层在1e15cm-3处被轻掺杂，以获得高量子效率和相对低的击穿电压。吸收层下面是一个缓冲层，设计为1μm，掺杂在1e18cm-3，该层的作用是提高外延质量和调节击穿电压。


2.制造：
通过在ICP高密度等离子刻蚀机中使用SF6和HBr气体进行刻蚀，形成深沟槽，镍作为刻蚀掩模通过磁控溅射和金属剥离得到，窗口的刻蚀也是通过ICP刻蚀进行的。通过PECVD方法同时形成沟槽边缘的钝化层和抗反射层，得到厚度为40nm的SiO2层。背面的n型接触金属由200nm的镍形成，顶部的p型接触由Ti（50nm）/ Al（190nm）形成，两者都在1000°C的RTA退火下进行2分钟。

通过矩形TLM方法测量了p型接触的欧姆接触性能，并将结果显示在图中。矩形电极之间的电阻随着电极间距的增加而增加，计算得到p-SiC上的具体接触电阻率为$5.9×10^{−4}Ω⋅cm^2$。

# Test and Analysis

1.研究了边缘钝化层的存在对暗电流的影响：
具有边缘钝化层的器件的暗电流比没有边缘钝化层的器件低1到2个数量级，证明了边缘漏电在器件的暗电流中起着重要作用，而具有良好质量的钝化层可以在一定程度上改善暗电流。
![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-3_edge%20passivation.png)

边缘漏电实质上是由被捕获载流子引起的边缘复合电流，与沟槽边缘的缺陷密度有关。通过PECVD在边缘处生长SiO2钝化层，可以在一定程度上填充沟槽边缘的自由价键，降低边缘处的缺陷密度。因此，如果采用热氧化方法生长SiO2，器件的边缘漏电将会更低。


2.测量器件的I-V特性和增益：
器件的击穿电压为127V；光增益接近1000

![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-4_iv%20%20and%20gain.png)

随着窗口深度的增加，器件的响应度增加，峰值值向较短的波长移动。

3.实测响应度与模拟结果的比较：
在260纳米波长处，响应度的峰值可以达到0.15A/W，对应的量子效率为71.5%。

![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-5_measurement%20and%20simulation.png)

4.带有和不带有抗反射层的器件之间的响应度差异：
40纳米厚的抗反射层可以提高响应度，而不改变峰值位置。

![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-6_anti-reflection%20layer.png)

5.不同电压下的响应:
当施加电压为21V时，器件的吸收层完全耗尽，其响应度高于5V时的响应度；当施加电压为110V时，响应度明显增加，并显示出一定的光增益；当施加电压为126V时，发生雪崩倍增过程，响应度迅速增加。

![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/media/JC177-7_responsivity%20under%20different%20bias%20voltage.png)

# Conclusions

设计了一种基于分离吸收和倍增结构的4H-SiC雪崩光电探测器。通过刻蚀深沟槽形成边缘终端，利用表面凹陷窗口提高量子效率。通过PECVD方法生长边缘钝化层，发现它在抑制沟槽漏电方面起着重要作用。器件的击穿电压为127V，增益接近1000个数量级。峰值响应度为0.15A/W，接近260nm波长，相应的量子效率为71.5%。
