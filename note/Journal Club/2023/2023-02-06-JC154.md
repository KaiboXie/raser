[[王浩博]] 

[Characterization of silicon microstrip detectors using an infrared laser system](https://doi.org/10.1016/S0168-9002(98)01337-0)

利用可变波长红外激光系统研究了硅微带探测器的基本特性。给出了有关相邻微条之间电荷共享的测量结果。在η函数的帮助下，在描述电荷共享的模型框架中解释了结果。

穿透深度$a=\frac{1}{\alpha}$
激光能量随深度y的损失可用指数表示，y处的激光能量的$E(y)=E_{0}\left ( 1-e^{-y/a}  \right )$
平均能量损失深度$\bar{A} =\frac{\int_{0}^{b}y\left ( dE/dy \right )  }{\int_{0}^{b}\left ( dE/dy \right ) } = a\left ( \lambda  \right )- b\tfrac{e^{-b/a( \lambda)} }{1-e^{-b/a( \lambda)}}$(b为探测器厚度280微米)
![](https://raser-1314796952.cos.ap-beijing.myqcloud.com/JC154-penetration-depth-in-silicon.png)
1050nm激光的$\bar{A}$与MIP相近，因此1050nm处产生的电荷与MIP的电荷沉积非常相似。

$\eta$函数表示相邻读出电极间的charge sharing
$\eta$为激光位中心$x_0$的函数，$PH_L,PH_R$表示相邻左右读出电极读出波形的高度
$\eta (x_{0})=\frac{PH_{R}(x_{0})}{PH_{R}(x_{0})+PH_{L}(x_{0})}$

$\frac{\mathrm{d} y}{\mathrm{d} t}=\mu _eE_{y}(y)$

读出条产生的信号用于重建初始粒子或激光束的位置。最常用的算法是计算the center of gravity for the collected charge。
如果只考虑两条条带，重建后即为$\eta$函数。其他条带的贡献在5%忽略不计。
图16激光B中n侧的结果，重构后的位置与图16b中已知的激光位置进行对比。
用直方图表示差异。r.m.s.为3.2 μm
探测器的数字分辨率$r.m.s._{digi}=54/\sqrt{12}=15.6\mu m$

不同参考文献对穿透深度的预测差异高达30%。这种差异可能是由所涉及的晶圆材料的差异造成的。不同的晶格类型和杂质浓度可能会影响光学性质。测得的穿透深度与预测值相差2-3个$\sigma$。通过计算charge sharing没有显著差异。
