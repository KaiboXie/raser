## 简介

- SPA：具有二维分辨率，沿光束方向均产生电子-空穴对，对沿光束方向的变化不敏感
- TPA：具有三维分辨率，仅在特定区域内产生电子-空穴对，实现硅探测器的三维表征
- 在CERN搭建了第一台比较完整的TPA-TCT测试系统的系统
- 对未辐照和辐照后的HPK探测器进行测试并得到结果

## TPA-TCT

单个电子被两个能量低于带隙的光子激发，发生的概率与光子浓度有关，认为仅发生在焦点附近。
$\frac{\mathrm{d} I\left ( r,z,t \right ) }{\mathrm{d} z} = -\alpha I\left ( r,z,t \right ) + \beta I\left ( r,z,t \right )^2$

$\alpha$:SPA吸收系数 ，单个光子能量低于带隙可忽略
$\beta$ :TPA吸收系数,与材料三阶极化率χ ( 3 )的虚部成正比

$\frac{\mathrm{d} n(r,z,t)}{\mathrm{d} t}=\frac{\beta}{2\hbar \omega}  I^{2}(r,z,t)$

后续实验验证
